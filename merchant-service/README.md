# Merchant Onboarding Microservice

## Tech Stack
- Java 17
- Spring Boot 3.5.3
- Spring Data JPA
- MySQL
- Swagger OpenAPI
- Validation API
- Maven

## Features
- Create Merchant
- Get Merchant By Id
- Get All Merchants
- Approve Merchant
- Request/Response DTO
- Global Exception Handling
- Input Validation
- Swagger Documentation

## Run

```bash
./mvnw spring-boot:run
Swagger:

http://localhost:8081/swagger-ui/index.html
